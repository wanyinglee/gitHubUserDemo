# GitHub User Repository Demo

- A mini demo retrieving GitHub users' repositories info

## Prerequisites

Required:
- Git
- NodeJS
- npm
- Yeoman

## Installation
- Setup npm globally: 
```bash
$ npm install npm -g
```
- Install Yeoman
```bash
$ npm install -g yo
```
-Install angular generator
```bash
$ npm install -g generator-angular
```

## Run
```bash
$ grunt serve
```
